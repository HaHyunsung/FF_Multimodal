# 추가 실험 계획 — 결과의 애매함 보강

> 현 결과의 약점: ① 융합 이득이 ±0.01로 노이즈 수준 ② 이미지 존재구간 F1 1.000이 비현실적(작고 편중된 테스트셋) ③ 통제실험이 이미지만 약화해 불완전.
> 아래 3개 실험으로 "멀티모달 이득"을 통계적으로 단단하게 만든다. (Kaggle GPU 재실행 필요)

---

## 문제 진단 (왜 애매한가)

| 약점 | 원인 | 근거 |
|------|------|------|
| Fusion 이득 미미 | 센서·이미지 둘 다 너무 강함(불균형) | FAIR: Fusion 0.8714 vs Sensor 0.8710 (+0.0004) |
| Image 1.000 비현실 | 이미지 테스트셋 작음(2,346장) + 클래스 편중(NoNose 73%, 3클래스만) | classification_report support |
| 통제 불완전 | 이미지만 약화, 센서는 0.974로 여전히 강함 | 약화 결과 Sensor 0.974 vs Image 0.775 |

---

## 실험 A — 균형 통제 (센서도 함께 약화) ★최우선

**목적**: 센서·이미지를 **비슷한 수준**으로 맞춘 뒤 융합 이득 측정 → 진짜 "균형" 조건.

**방법**:
- 센서 BiLSTM을 약화: hidden 128→32, 또는 학습 3~5에폭으로 제한 → 센서 F1을 ~0.80대로
- 이미지도 동일하게 ~0.80대로 (현 약화 3에폭=0.775와 비슷)
- 두 모델 모두 0.80 부근일 때 Decision-Fusion 비교

**코드 스케치** (노트북 추가 셀):
```python
# 약한 센서 모델
sensor_weak = SensorLSTM(input_dim=NUM_SENSOR_FEATURES, hidden_dim=32).to(DEVICE)
run_training(sensor_weak, train_sensor_loader, test_sensor_loader,
             "sensor", "sensor_weak", epochs=3)
# 이미지 존재 구간에서 sensor_weak + image_weak 확률로 decision fusion
# → Sensor_weak / Image_weak / Fusion 비교 (셋 다 ~0.8 균형)
```
**기대**: 둘 다 0.8 균형이면 Fusion이 0.85+로 명확히 초과 → 멀티모달 이득 깔끔 입증.

---

## 실험 B — 더 크고 어려운 이미지 테스트셋

**목적**: 이미지 1.000(편중·과쉬움)을 현실적 난이도로.

**방법**:
- 이미지 Part 2~6 추가 (현재 Part 1만) → 클래스 균형↑, 표본↑
- cycle 4·9 전체에서 5클래스 모두 등장하도록
- 이미지 단독 F1이 0.85~0.95로 내려오면 더 신뢰성 있는 비교

**주의**: Part 추가 시 데이터 다운로드/매칭 시간. `dataset_sources`에 part 추가하면 코드가 자동 매칭(KAGGLE_IMAGE_ROOTS glob).

---

## 실험 C — 통계적 유의성 검증

**목적**: ±0.01 차이가 우연인지 검증.

**방법**:
```python
import numpy as np
seeds = [42, 7, 123, 2024, 999]
fusion_f1s, sensor_f1s = [], []
for sd in seeds:
    torch.manual_seed(sd); np.random.seed(sd)
    # 재학습 또는 test 부트스트랩 재샘플링
    ...
print(f"Fusion {np.mean(fusion_f1s):.4f} ± {np.std(fusion_f1s):.4f}")
print(f"Sensor {np.mean(sensor_f1s):.4f} ± {np.std(sensor_f1s):.4f}")
# paired t-test
from scipy.stats import ttest_rel
print(ttest_rel(fusion_f1s, sensor_f1s))
```
- 또는 **test set 부트스트랩**(재학습 없이 빠름): 테스트 샘플을 1000회 복원추출해 F1 분포 → 신뢰구간.
**기대**: Fusion>Sensor가 신뢰구간 밖이면 유의, 겹치면 "차이 없음" 솔직 인정.
