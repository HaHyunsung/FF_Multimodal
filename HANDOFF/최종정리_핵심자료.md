> FF 멀티모달 이상탐지 · 3조(김주헌·하현성) · 2026-06-09
> **기준 실행 = iter6 (이미지 LR 1e-4 안정화) + 열화 견고성 실험.** 이 문서의 숫자만 쓰면 안 헷갈림.

## 0. 한 줄 결론
> **멀티모달 융합은 무조건 더 정확한 게 아니라, 이미지가 나빠질 때 단독보다 안정적이다.**
> 깨끗한 이미지에선 이미지 단독이 이미 강해 융합 이득이 작지만, 이미지가 나빠지면(실제 공장 조건) 단독은 무너지고 적절한 융합이 성능을 지킨다. **단, 어떤 융합 방식이냐가 결정적** — 고정 가중 융합은 취약, 적응형(주의) 융합은 견고. → **신뢰도 기반 적응형 가중치**가 핵심 방향.

---

## 1. 데이터·모델 핵심 사실
- **데이터**: Future Factories(USC), 166,001 시점, 285 사이클, 5클래스(Normal 54.7% 등). 시점 단위 분류.
- **센서**: 22채널, **모든 시점**에 존재 → BiLSTM(양방향 2층).
- **이미지**: 카메라 2대(한 시점=2장). 카메라는 **전 구간 촬영**하나 부품이 보이는 **Cycle 4·9만 선별 사용**. 6파트 중 **2파트(~200GB)** 사용 → 두 카메라 매칭 **19,234 시점**.
- **융합**: Decision-level(확률 가중) / Concat(특징 결합, 학습형) / Cross-Attention(적응형 주의). 이미지 결측 시 센서 폴백.
- **분할**: cycle-wise 80/20 (누수 방지). 지표 **Weighted F1**.
- **평가셋 3종**:
  - 전체 cycle 4·9 테스트 = **9,521 시점**(FAIR 비교셋)
  - 그중 이미지 실제 존재 = **3,912 시점(41%)** ← 단독 vs 융합 정면 비교 가능한 유일 구간
- **강의기법 매핑**: CNN·전이학습(7-2)→ResNet18 / RNN·LSTM(10-2)→BiLSTM / 멀티모달(9-1)→Fusion.

## 2. 실험 목록 — [왜] [무엇] [결과] [의미]

### 실험 A. 세 모델 기본 비교 (FAIR, 9,521 시점)
- **왜**: 모델마다 가용 범위가 달라, 동일 테스트셋에서 공정 비교 필요.
- **결과**: Sensor 0.871 / Image 0.826 / Concat 0.900 / CrossAttn 0.871 / Decision 0.893.
- **의미**: 전체 구간은 이미지 결측이 많아(59%) 융합 이득이 희석됨. → 이미지 존재 구간을 따로 봐야 함.

### 실험 B. 이미지 존재 구간 (3,912 시점) — 멀티모달 이득 검증
- **왜**: 이미지가 실제 있는 곳에서만 "이미지를 더한 효과"가 드러남.
- **결과**: Sensor 0.907 / Image 0.957 / **Concat 0.977** / CrossAttn 0.906 / Decision 0.957.
- **의미**: 모든 시점에 이미지가 있는 이 구간에선 **Concat이 두 정보를 비선형 결합해 최강(0.977)**. Decision은 0.957로 이미지와 동률(단순 가중평균이라 최고 단독을 못 넘음). 부트스트랩 **Decision−Sensor +0.050, 95% CI [+0.039,+0.060] 유의**. 이미지가 이미 강해 정확도 이득 자체는 작음(+0.020). ※ 결측 포함 전체 셋(실험 A)에서도 Concat 0.900 ≥ Decision 0.893 → **Concat이 전 구간 최고이며, "결측 환경에서 Decision이 더 낫다"는 근거는 없음.**

### 실험 C. 중간발표 → 최종 (데이터 균형 가설)
- **왜**: 중간발표 때 융합이 졌음(Fusion 0.88 < Sensor 0.93). 원인=데이터 불균형 가설.
- **무엇**: 이미지 1→2파트 + 카메라 1→2대 확장, 융합을 단순 Concat→Decision 등으로 개선.
- **결과**: 이미지 존재 구간에서 융합이 단독을 넘김(가설 검증).
- **의미**: 멀티모달 성공은 데이터 균형·가용성에 의존.

### 실험 D. 이미지 학습 안정화 (V13 → iter6)
- **왜**: V13에선 이미지 학습이 불안정해 Normal·NoBody1 = 0.00(붕괴), 이미지 F1 0.80.
- **무엇**: 이미지 학습률 1e-3 → **1e-4** (ResNet18 전체 fine-tune엔 1e-3이 과대).
- **결과**: 이미지 학습 안정화, 모든 클래스 회복, **이미지 F1 0.80 → 0.96** (논문 ~0.97과 일치).
- **의미**: 정직한 강한 이미지 확보. 단, 이미지가 강해져 융합의 *정확도* 이득은 축소 → "견고성"으로 가치 재정의 필요.

### 실험 E. ⭐이미지 열화 견고성 (핵심 발견)
- **왜**: 깨끗한 실험실 이미지는 0.96으로 너무 강함. 실제 공장은 조명·가려짐·노이즈로 이미지가 나빠짐(팀의 원래 동기). 그 조건에서 누가 버티나?
- **무엇**: 학습된 모델 그대로, **테스트 이미지만 점점 열화**(블러+조명변화+가림)시켜 추론만 비교.
- **결과** (이미지 존재 3,912 시점):

| 이미지 품질 | Image | Concat | Decision | CrossAttn | Sensor |
|---|---|---|---|---|---|
| 깨끗 | 0.957 | **0.977** | 0.957 | 0.906 | 0.907 |
| 중간열화 | 0.681 | **0.931** | 0.775⤵ | 0.905 | 0.907 |
| 강열화 | 0.028 | 0.303⤵ | 0.116⤵ | **0.905** | 0.907 |

(⤵ = 센서 0.907보다 추락)
- **의미 (3가지)**:
  1. **Decision-Fusion(고정 가중)이 가장 취약** — 중간 열화만으로 센서 밑(0.775). 고정 w=0.55가 나쁜 이미지를 계속 믿어서.
  2. **Concat(학습형)은 중간 열화까지 버팀(0.931)** 후 강열화에 붕괴.
  3. **CrossAttn(적응형 주의)은 끝까지 견고(0.905)** — 단 깨끗할 때도 ≈센서라 이미지 이득은 못 챙김.
  - → **어떤 융합도 모든 조건에서 최고가 아님.** 신뢰도 기반 적응형 가중치가 답.
- **정직한 주의**: 강열화(이미지 0.03)는 "거의 파괴" 수준의 극단 가정. "극단적 열화 하에서"로 명시.

### (보조) 결함 유형별 (Concat, cycle 4·9)
NoNose 0.97 / NoNose_NoBody2 0.94 / **3부품 0.71(가장 어려움)** / Normal 0.92.

---

### 융합 방식별 강점 — 요약
| 융합 | 특성 | 약점 |
|---|---|---|
| **Concat** (학습형 특징결합) | **정확도 전 구간 최고** (이미지존재 0.977, 결측 포함 전체 0.900) | 심한 열화엔 붕괴(0.30) |
| **Decision** (고정 가중 + 폴백) | 이미지 없으면 센서 폴백 → **결측에 안정적**(단독 상회), 단순함 | 정확도는 Concat보다 낮음(0.957/0.893), 열화에 취약 |
| **CrossAttn** (적응형 주의) | **이미지 열화에 견고**(0.90 유지) | 깨끗할 때 이미지 이득은 못 챙김(≈센서) |

→ **정확도는 Concat이 전 구간(이미지존재·결측포함 모두) 최고, 견고성은 CrossAttn.** Decision은 단순·결측 안정 baseline(승자 아님). ※ "결측 많을 때 Decision이 더 낫다"는 근거 없음 — 전체 셋에서도 Concat 0.900 ≥ Decision 0.893.

## 3. 최종 종합 결론 (보고서 결론에 그대로 사용 가능)
1. **깨끗한 데이터**: 이미지 단독이 0.96으로 매우 강해, 융합의 정확도 이득은 작음(Concat 0.977, +0.02).
2. **융합은 이미지 품질이 나쁠 때 더 유용**: 이미지 열화 시 단독은 붕괴, 적절한 융합은 성능 유지.
3. **융합 전략이 결정적**: 정확도는 Concat이 전 구간(이미지존재 0.977·결측포함 전체 0.900) 최고, 열화 견고성은 적응형(CrossAttn)만 확보. Decision은 결측엔 안정적이나 정확도는 Concat보다 낮고 열화에 취약(센서 밑 추락).
4. **결측엔 설계상 견고**(센서 폴백). **열화엔 적응형 가중치 필요**(미해결 → 향후 핵심).
5. **의의**: "멀티모달이 무조건 좋다"가 아니라 "**언제·어떤 방식으로 좋은가**"를 정량 규명. 이미지가 나빠지기 쉬운 환경에서는 적응형 융합이 더 유용.
